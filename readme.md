# Simple Graphics Manager

Приложение для построения одного или нескольких графиков на одном чертеже с расширенными возможностями настройки и сохранения результатов.

![alt text](image-1.png)

## Основные возможности
- **Множественные графики** - до 15 функций на одном чертеже
- **Гибкая настройка** - индивидуальные параметры для каждого графика
- **Ссылки между функциями** - возможность использовать другие функции в вычислениях
- **Интерактивный интерфейс** - визуальная обратная связь при ошибках ввода
- **Экспорт результатов** - сохранение чертежа в формате PNG

Приложение написано на Python с использованием библиотек PyQt5, Numpy, Matplotlib

## Как пользоваться
### **1. Добавление функций**
При запуске отображается окно со строкой ввода формулы
Нажмите "Добавить" для создания новых полей ввода (максимум 15)
Используйте кнопку "Создать" для построения всех графиков

*Синтаксис формул*
Поддерживается Python-синтаксис со следующими дополнениями:
Математические операции:
Стандартные: +, -, *, /
Возведение в степень: ** или ^
Десятичные дроби: . или ,

*Функции и константы*
Тригонометрия: sin(x), cos(x), tan(x), arcsin(x), arccos(x), arctan(x)
Логарифмы: log(x) (натуральный), lg(x) (десятичный)
Другие: exp(x), sqrt(x)
Константы: e, pi

### **2. Управление графиками**
Для каждой функции доступно:

- Скрыть/показать (галочка слева)
- Удалить (кнопка с крестиком справа)
- Настроить (цветной квадрат справа)

### **3. Ссылки между функциями**
Используйте имена функций (например, f0 или f12) в других формулах.
Ограничения:
Функции нельзя ссылаться на саму себя
Функции нельзя использовать функции с ошибками
Запрещены циклические ссылки

### **4. Настройки графика**

*Через меню кастомизации (цветной квадрат) доступно:*

![alt text](image-2.png)

- **Цвет** - настройка RGB и прозрачности
- **Тип графика** - непрерывный, точечный, диаграмма
- **Точность** - шаг построения (особенно важно для точечных графиков)
- **Смена осей** - галочка x(y) для отображения функции как x от y
- **Диапазон значений** - ручная настройка масштаба
- **Названия** - подписи для графика и осей

*Для общих настроек графика:*
- **Сетка** - включение/отключение координатной сетки
- **Автокомпрессия** - график растягивается под размер поля 
(иначе фиксированный диапазон [-15;15])

### **5. Сохранение результатов**
Нажмите кнопку "Save" для сохранения чертежа:
Открывается диалоговое окно выбора файла
Сохранение в формате PNG
Сохраняется текущее состояние всех видимых графиков


Приложение идеально подходит для учебных целей, быстрой визуализации математических функций и анализа зависимостей между различными графиками.
